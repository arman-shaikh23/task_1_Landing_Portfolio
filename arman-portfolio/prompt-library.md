# Prompt Library for Portfolio Website Development

Here are 10 examples of good and bad prompts for modifying your portfolio website (HTML/CSS/JS). Using the bad prompt may lead to unexpected or incorrect results, while the good prompt will help you achieve the desired outcome.

---

### 1. **Bad Prompt**: "Make the header look better"
   **Good Prompt**: "Change the header background color to #333, set the text color to white, and increase the padding to 20px on all sides."

### 2. **Bad Prompt**: "Fix the menu"
   **Good Prompt**: "In the navigation menu, change the hover state of links to have a background color of #007bff and text color of white, and remove the underline."

### 3. **Bad Prompt**: "Make the image bigger"
   **Good Prompt**: "In the about section, set the width of the profile image to 200px and height to auto, and add a border-radius of 10px."

### 4. **Bad Prompt**: "Center the text"
   **Good Prompt**: "In the projects section, set the text-align of project titles to center and add a margin-top of 15px."

### 5. **Bad Prompt**: "Add a button"
   **Good Prompt**: "After the introductory paragraph in the home section, add a button with the text 'View My Work', set its background color to #28a745, text color to white, padding to 10px 20px, and border-radius to 5px."

### 6. **Bad Prompt**: "Make it responsive"
   **Good Prompt**: "Add a media query for screens max-width: 768px that changes the flex-direction of the projects container to column and reduces the padding to 10px."

### 7. **Bad Prompt**: "Fix the font"
   **Good Prompt**: "Set the font-family of the entire body to 'Arial, sans-serif', and for headings (h1, h2, h3) use 'Georgia, serif'."

### 8. **Bad Prompt**: "Add a shadow"
   **Good Prompt**: "Add a box-shadow to the project cards: 0 4px 8px rgba(0,0,0,0.1), and on hover increase it to 0 6px 12px rgba(0,0,0,0.15)."

### 9. **Bad Prompt**: "Make the form work"
   **Good Prompt**: "In the contact form, add an event listener to the submit button that prevents the default form submission, validates that the name and email fields are not empty, and shows an alert if validation fails."

### 10. **Bad Prompt**: "Change the color scheme"
    **Good Prompt**: "Update the CSS variables: set --primary-color to #007bff, --secondary-color to #6c757d, and --background-color to #f8f9fa. Then replace all hardcoded colors with these variables."
