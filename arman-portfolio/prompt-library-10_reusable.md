# 10 Reusable Prompt Templates for Portfolio Website Development

These templates are designed to be reused for common modifications to your HTML/CSS/JS portfolio website. Simply replace the text in `[brackets]` with your specific details.

---

### 1. **Style Element Property**
   **Template**: "Change the [selector] [CSS property] to [value]."
   **Example**: "Change the header background-color to #333."

### 2. **Adjust Element Size**
   **Template**: "Set the [selector] width to [value] and height to [value]."
   **Example**: "Set the .profile-img width to 200px and height to auto."

### 3. **Add Spacing**
   **Template**: "Add [top/right/bottom/left]-[margin/padding] of [value] to the [selector]."
   **Example**: "Add bottom-margin of 20px to the .project-card."

### 4. **Change Font Properties**
   **Template**: "Set the [selector] font-family to '[font]', font-size to [value], and font-weight to [value]."
   **Example**: "Set the h1 font-family to 'Georgia, serif', font-size to 2.5rem, and font-weight to bold."

### 5. **Make Responsive Adjustment**
   **Template**: "Add a media query for [condition] that changes [selector] [property] to [value]."
   **Example**: "Add a media query for max-width: 768px that changes the .projects-container flex-direction to column."

### 6. **Add Hover Effect**
   **Template**: "On hover, change the [selector] [property] to [value] and [property] to [value]."
   **Example**: "On hover, change the .nav-link background-color to #007bff and color to white."

### 7. **Center Element**
   **Template**: "Center the [selector] horizontally using [method] and vertically using [method]."
   **Example**: "Center the .content horizontally using margin: 0 auto and vertically using flexbox on its parent."

### 8. **Add Visual Effect**
   **Template**: "Add [effect type] to the [selector]: [value]."
   **Example**: "Add box-shadow to the .card: 0 4px 8px rgba(0,0,0,0.1)."

### 9. **Update Text Content**
   **Template**: "Change the text content of the [selector] to '[new text]'."
   **Example**: "Change the text content of the #intro-paragraph to 'Welcome to my portfolio!'."

### 10. **Toggle Visibility**
    **Template**: "Set the [selector] display property to [value] to [show/hide] it."
    **Example**: "Set the #mobile-menu display property to block to show it on mobile devices."
